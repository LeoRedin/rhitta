# Conventions

The canonical conventions document for Rhitta lives at the repo root: [`AGENTS.md`](../../AGENTS.md).

Architectural decisions that justify those conventions live in [`docs/adr/`](../adr/).

This folder is reserved for **deep-dive convention docs** that don't fit in AGENTS.md without bloating it — things like the module DI pattern's full file-by-file shape, the import-boundary rules per layer, naming conventions for repositories, etc. Phase 0 ships AGENTS.md and the ADRs; deep-dives land alongside the packages they describe.
